Logistic Inteligente
Descripción

Este proyecto es una presentación en formato página web que muestra cómo se ha mejorado la logística mediante la optimización de rutas, la gestión de datos y el cálculo de cargas de camiones. Hemos utilizado nuestros conocimientos en business para que la página muestre KPIs importantes y compare nuestra solución con la solución que había antes.

Archivos del proyecto
DAMM_LogisticaIInteligente.html

Es la página principal del proyecto. Explica de forma visual cómo se ha optimizado la flota de transporte y las rutas para mejorar la eficiencia logística. También muestra KPIs importantes y compara la solución propuesta con la situación anterior.

DAMM_MOLLET.html

Contiene los datos ya depurados. Muestra el proceso de limpieza de datos aplicado a un caso real en Mollet.

PAGINA PARA CALCULAR LA CARGA.html

Es una herramienta que permite subir un archivo Excel con las comandas y calcular cómo organizar la carga del camión de forma eficiente.
